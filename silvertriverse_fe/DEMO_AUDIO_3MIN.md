# SilverTriverse 3-Min Audio Demo

This demo mode is mapped to your narration flow and runs about 3 minutes.

## Terminal commands

- Install dependencies (first time):
  - `npm install`
- Run normal app:
  - `npm run dev`
- Run auto demo mode (3-min audio script):
  - `npm run demo:audio3min`
- Run default demo mode (existing longer sequence):
  - `npm run demo:default`

## URL option

If `npm run dev` is already running, open:

- `http://localhost:5173/demo.html?demoScript=audio3min`

## Scene order for audio sync

1. Citizen Entry (`/reelity`)  
2. Reelity Social Engine (`/reelity`)  
3. Land as Identity (`/land`)  
4. Avatar Assistants (`/ai-producer`)  
5. Collectible Units (`/collectible-units`)  
6. Utility to Township (`/land-world`)  
7. Economy & Trading (`/relics`)  
8. Visionary Finale (`/`)

## Recording notes

- Keep browser zoom at 100%.
- Start audio first, then start demo immediately.
- Press `Esc` to stop demo at any time.
